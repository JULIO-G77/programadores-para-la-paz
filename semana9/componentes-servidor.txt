Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar programas y gestionar el hardware.

Node.js:
Es un entorno de ejecución que permite ejecutar JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene la lógica del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
