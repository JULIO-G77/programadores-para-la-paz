 conceptos-git.txt

Repositorio:

Un repositorio es un espacio donde se almacenan los archivos y cambios de un proyecto. Permite organizar la información y trabajar de forma colaborativa utilizando herramientas como Git y GitHub.

Commit:

Un commit es un registro de los cambios realizados en un proyecto. Cada commit guarda una versión específica del trabajo realizado para mantener un historial organizado.

Push:

El comando push permite enviar los cambios realizados desde el computador local hacia el repositorio remoto en GitHub. Esto ayuda a mantener el proyecto actualizado y respaldado.
